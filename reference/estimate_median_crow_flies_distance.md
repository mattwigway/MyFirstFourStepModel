# Estimates the median crow-flies distance from the NHTS in kilometers, by dividing the horse-flies distance by 1.3. This is based on the paper https://arxiv.org/abs/2406.06490

Estimates the median crow-flies distance from the NHTS in kilometers, by
dividing the horse-flies distance by 1.3. This is based on the paper
https://arxiv.org/abs/2406.06490

## Usage

``` r
estimate_median_crow_flies_distance(nhts)
```
