# Get the time period from an NHTS time. Currently these are hardcoded as AM Peak 6-10 AM, Midday 10-4, PM Peak 4-7, Overnight 7PM - 6AM.

Get the time period from an NHTS time. Currently these are hardcoded as
AM Peak 6-10 AM, Midday 10-4, PM Peak 4-7, Overnight 7PM - 6AM.

## Usage

``` r
get_time_period(time)
```
