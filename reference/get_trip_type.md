# This returns the trip type for each trip in an NHTS trip table. It will classify trips as HBW - Home based work HBO - Home based other NHB - Non-home-based

This returns the trip type for each trip in an NHTS trip table. It will
classify trips as HBW - Home based work HBO - Home based other NHB -
Non-home-based

## Usage

``` r
get_trip_type(whyfrom, whyto)
```
